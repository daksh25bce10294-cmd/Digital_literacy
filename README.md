🧩 Task 1: Digital Literacy Awareness Infographic
📌 Description
As part of the Digital Literacy project, I created a one-page infographic to explain the concept of digital literacy and its importance for students. The infographic is designed to be simple, informative, and visually engaging so that it can be easily understood by my peers.

 Tool Used
Canva
I used Canva, an online design tool, to create the infographic using templates, icons, and visual elements.
📊 Topics Covered
The infographic includes the following key areas:
What is Digital Literacy – Understanding the ability to use digital tools safely and effectively
Useful Digital Tools for Students – Tools that help in learning, productivity, and collaboration
Safe Internet Practices – Tips for staying safe online such as using strong passwords and avoiding suspicious links
🎯 Objective
The main objective of this task was to create awareness about digital literacy and help students understand how to use technology responsibly and efficiently.
💡 Learning Outcome
Through this task, I learned how to present information in a visual format and gained basic design skills. It also improved my understanding of digital literacy and its importance in daily life.
